# 🚀 Getting Started

Clone Repository

```bash
git clone https://github.com/ankit-2003-star/Ankit-Anand-portfolio.git
```

Install Dependencies

```bash
npm install
```

Run Development Server

```bash
npm run dev
```

Create Production Build

```bash
npm run build
```

Preview Production Build

```bash
npm run preview
```
